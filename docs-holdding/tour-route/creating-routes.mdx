# Creating Tour Routes

## Overview
Learn how to create and optimize your tour routes using Band Practice CRM's route planning features.

## Adding Stops
1. Navigate to the Tour Route section
2. Use the venue search to find existing venues
3. Or manually enter new venue details:
   - Venue name
   - Street address
   - City
   - State
   - ZIP code

## Managing Your Route
- Drag and drop stops to reorder
- View total mileage calculations
- See distance between venues
- Export route details to PDF

## Smart Features
- Automatic date suggestions based on routing
- Warning system for tight scheduling
- Interactive map visualization
- Real-time distance calculations
