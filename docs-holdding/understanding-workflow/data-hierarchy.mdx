# Data Hierarchy

Learn how Band Practice CRM organizes and structures your data for efficient management.

## Overview

Band Practice CRM uses a hierarchical data structure to organize your touring and booking information effectively. Understanding this structure helps you manage your data more efficiently.

## Core Data Structure

### Primary Entities
- **Tours**
  - Tour name
  - Date range
  - Geographic scope
  - Tour type
- **Gigs**
  - Show dates
  - Venues
  - Performance details
  - Technical requirements

### Supporting Data
- **Venues**
  - Location info
  - Technical specs
  - Contact details
  - Performance history
- **Contacts**
  - Venue staff
  - Promoters
  - Technical crew
  - Support services
