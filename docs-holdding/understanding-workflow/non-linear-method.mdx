# Non-Linear Method

Learn how Band Practice CRM's non-linear workflow adapts to your unique booking and touring process.

## Overview

The non-linear method allows you to work on different aspects of tour planning simultaneously, adapting to changing circumstances and opportunities. This flexible approach helps you manage complex touring schedules effectively.
