# Gig Input Form

Learn how to create and edit gigs in Band Practice CRM.

## Creating a New Gig

1. Click "New Gig" button
2. Fill in essential details:
   - Date and time
   - Venue
   - Payment terms
   - Technical requirements
3. Save your gig

## Required Fields

- Venue name
- Date
- Set time
- Load-in time
- Payment amount
