# Application Elements

Learn about the key elements of Band Practice CRM's interface.


## Main Dashboard

- Standard 30,000ft view of your tour data
- Quick access to common tasks
- Recent activity feed
- Real Time Analytics Data

## Navigation

- User Badge: Account information.  Located a the top of the left sidebar.
- Left sidebar: Main menu
- Footer Links: Context-specific information

## Common Elements

- Tour Route Map
- Saved Data
- Saved Venues and Leads
