# Gig Based Routing

Learn how to create efficient routes based on your confirmed gigs.

## Route Optimization

The system helps you:
- Calculate optimal driving routes
- Estimate travel times
- Plan fuel stops
- Schedule rest periods

## Using the Route Planner

1. Select your tour
2. Choose gigs to include
3. Set preferences:
   - Preferred driving hours
   - Required rest stops
   - Maximum daily distance
4. Generate optimized route
