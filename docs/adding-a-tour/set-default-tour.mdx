# Set a Default Tour

Learn how to set up a default tour for easier gig management.

## Why Set a Default Tour?

Setting a default tour helps streamline the process of adding new gigs by:
- Automatically associating new gigs with the current tour
- Maintaining consistent tour settings
- Reducing data entry time

## Setting Up a Default Tour

1. Navigate to Tour Management
2. Select or create a new tour
3. Click "Set as Default"
4. Configure default settings:
   - Tour name
   - Date range
   - Default routing preferences
